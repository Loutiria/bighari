# Bighari PMP-Aligned List System Rebuild

This build corrects the list-layout issue across the prototype while keeping the system aligned with the PMP document.

## Main changes

- Replaced the Order Status long list with grouped collapsible status sections.
- Rebuilt Order Tracking into simple workflow groups.
- Converted inventory rows into compact material strips.
- Grouped stock availability by Stable, Low Stock, Out of Stock, and Incoming Restock.
- Simplified alerts into Critical, Warning, and Info strips.
- Kept reports simple, printable, and basic as required by the PMP scope.
- Kept delivery/order completion simple without GPS or logistics tracking.
- Reduced visual clutter, repeated labels, and oversized cards.

## Scope alignment

Included:
- order recording and monitoring
- order status tracking
- inventory monitoring
- material categorization by color
- stock availability checking
- low-stock alerts
- mobile-friendly UI
- basic reports

Excluded:
- AI forecasting
- GPS delivery tracking
- accounting or payroll
- enterprise integrations
- multi-branch management
